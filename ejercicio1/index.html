  • number:0 (a la izquierda el valor, a la derecha el número de decimales) 4238.87437 -> {{4238.87437 | number:0 }}
  • number:2 4238.87437 -> {{4238.87437 | number:2 }}
  • date:'medium' (timestamp | formato) 1299328823006 -> {{ 1299328823006 | date:'medium' }}
  • date:'dd/MM/yyyy' (timestamp | formato) 1299328823006 -> {{ 1299328823006 | date:'dd/MM/yyyy' }}
  • currency: € ( valor | currency: formato) 2000 -> {{ 2000 | currency: '€ ' }}
  • currency: € : numDecimales ( valor | currency: formato : decimales) 2000 -> {{ 2000 | currency: '€ ': 0 }}
